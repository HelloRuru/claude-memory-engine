# /訂正 — 翻錯題本

掃一遍錯題本，提醒自己之前哪裡做錯過。

## 步驟

1. **載入錯題本** — 讀取 `~/.claude/skills/learned/writing-review-list.md`
2. **過濾有效的** — 跳過標記 `cleared` 的，只看 `active` 的
3. **分類整理** — 依類型分組（語氣、結構、用詞、格式等）
4. **列出摘要** — 每條有效規則列出漏掉次數和最近的例子
5. **標記常犯的** — 漏掉 3 次以上的標記「需要升級成硬規則」
6. **回報** — 「N 條有效訂正。最常犯的：[列出漏最多次的]。」

## 什麼時候用

- 每次任務前**自動**跑一次（不用打指令）
- 想自己翻？隨時打 `/訂正` 就好，不用等任務、不用等迴圈
