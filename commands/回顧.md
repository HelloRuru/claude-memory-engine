# /回顧 — 反思日記

從這次對話產生一篇反思日記，記錄做了什麼、學到什麼、發現什麼規律。

## 步驟

1. **回顧這次對話**：列出使用者的主要需求、解決了什麼問題、改了哪些檔案
2. **寫日記條目**：存到 `~/.claude/sessions/diary/` 目錄，用日期命名
3. **格式如下**：

```markdown
# 日記 {YYYY-MM-DD}

## 今天做了什麼
- {列出主要工作項目}

## 學到的事
- {踩坑經驗、新發現、使用者的偏好}

## 觀察到的規律
- {使用者的工作模式、重複出現的需求}

## 下次要注意
- {提醒未來的 Claude}
```

4. **回報**：簡短說「日記寫好了，記了 X 個重點」

## 注意事項
- 用白話文，不要技術術語
- 重點是「學到什麼」和「規律」，不是流水帳
- 每篇控制在 30 行以內
