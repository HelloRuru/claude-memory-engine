# /備份 — 推記憶到 GitHub

把本地記憶檔案同步到 GitHub 備份。

## 步驟

1. **掃描本地記憶** — 讀取 `~/.claude/projects/{目前專案}/memory/` 所有 `.md` 檔案
2. **抓遠端 SHA** — 用 `gh api` 取得每個檔案最新的 SHA（每次都要抓新的，不用快取）
3. **比對差異** — 只推送有變更的檔案
4. **推送更新** — 用 `gh api` PUT 更新到 GitHub
5. **驗證** — 確認推送成功，列出更新了哪些檔案
6. **回報** — 簡短列出備份結果

## 注意事項

- 每個檔案都要先抓最新 SHA 再更新，避免衝突
- 非 ASCII 檔名用 URL encoding 處理
- GitHub API 回傳錯誤就回報，不要重試超過 3 次
- SessionEnd hook 會自動做本地 commit，這個指令負責 push
