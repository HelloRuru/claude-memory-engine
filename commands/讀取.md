# /讀取 — 把記憶載入目前的對話

SessionStart hook 自動注入簡短摘要，但 `/讀取` 是做**完整讀取** — 把記憶檔案的實際內容載入 context，讓 Claude 能引用細節。

## 步驟

1. **讀 MEMORY.md** — 掃描所有索引條目（通常已自動載入，確認一下）
2. **讀待辦檔案** — 載入 `memory/todo-status.md` 了解目前待辦
3. **讀最近變更** — 載入 24 小時內更新的檔案（最多 3 個）
4. **回報** — 「讀完了。{N} 個索引、{M} 項待辦、最近改了 {摘要}」

## 指定主題載入

如果使用者指定主題：
- 只載入該主題相關的檔案
- 例：「載入 blog」→ 只讀 blog 相關記憶檔

## 注意事項

- 這是**讀取**操作，不修改任何檔案
- 適合切換主題時載入詳細上下文
- 長對話中早期 context 被壓縮後，用這個重新載入
