# /同步 — 記憶雙向同步

把本地記憶變更推到遠端備份，或從遠端拉最新版本。

## 推送（預設）

1. 執行備份腳本：`bash ~/.claude/scripts/hooks/memory-backup.sh push`
2. 回報：推了什麼變更、推到哪個 repo

## 拉取

如果使用者說「同步」或「pull」：

1. 在記憶 repo 執行 `git pull`
2. 有衝突的話列出來，問使用者怎麼處理
3. 回報：拉了什麼、有沒有衝突

## 注意事項

- SessionEnd hook 每次對話結束會自動做本地 commit
- 這個指令負責**推到遠端**（GitHub 等）
- 推送 = 單向上傳；同步 = 雙向（推 + 拉）
