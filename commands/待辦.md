# /待辦 — 跨專案待辦追蹤

顯示所有專案的未完成項目，建議下一步該做什麼。

## 步驟

1. **讀待辦檔案** — 載入 `memory/todo-status.md`（或你設定的待辦追蹤檔）
2. **列出未完成項目** — 依專案分組，顯示狀態
3. **建議下一步** — 依急迫程度排序
4. **依專案篩選** — 如果使用者指定專案名稱，只顯示該專案的待辦

## 輸出格式

```
待辦事項

[專案 A]
- [ ] 任務描述（新增日期 {date}）
- [ ] 任務描述（新增日期 {date}）

[專案 B]
- [ ] 任務描述（新增日期 {date}）

建議下一步：{最緊急的任務}，因為 {原因}
```

## 注意事項

- 輸出要好掃描，不要長篇大論
- 沒有待辦就說沒有，問要不要新增
- 可以直接編輯待辦檔案來新增、完成或移除項目
