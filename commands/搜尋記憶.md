# /搜尋記憶 — 用關鍵字搜尋記憶檔案

跨所有記憶檔案搜尋關鍵字。

## 用法

`/搜尋記憶 {關鍵字}`

## 步驟

1. **確認搜尋範圍**：掃描以下目錄的 `.md` 檔案：
   - `~/.claude/projects/{目前專案}/memory/`
   - `~/.claude/sessions/diary/`
   - `~/.claude/skills/learned/`
   - `~/.claude/sessions/`（session 摘要）

2. **搜尋**：用 Grep 在所有 `.md` 檔案中搜尋關鍵字

3. **回報結果**：

```
記憶搜尋：「{關鍵字}」

找到 {N} 筆結果，分布在 {M} 個檔案：

{檔名}:{行號} — {符合的行預覽}
{檔名}:{行號} — {符合的行預覽}
...
```

4. 沒有結果的話：建議相關關鍵字或更廣泛的搜尋詞

## 注意事項
- 不分大小寫
- 最多顯示 20 筆結果，依最近修改的檔案優先排序
- 過長的行截到 120 字元
- 如果關鍵字是中文，也搜尋相關詞
